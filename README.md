# SwordBlock

A Fabric client-side mod for Minecraft 1.21.10 and 1.21.11 that brings back the classic **1.8 sword blocking animation**.

---

## Features

- Classic 1.8 sword blocking pose in **first person**
- Arm rotation in **third person** matching the original 1.8 look
- Automatically **disabled in Hypixel Bedwars** so it doesn't interfere with gameplay
- On **Hypixel**, suppresses the swing and hit animation while blocking (just like 1.8)
- Everywhere else, purely **visual** — vanilla mechanics are untouched
- Lightweight, client-side only — safe to use on any server

---

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.21.10 or 1.21.11
2. Install [Fabric API](https://modrinth.com/mod/fabric-api)
3. Download the latest `.jar` from [Releases](https://github.com/elyy442/SwordBlock/releases)
4. Drop it into your `.minecraft/mods` folder

---

## Usage

| Command | Description |
|---|---|
| `/swordblock` | Show current status |
| `/swordblock on` | Enable sword blocking |
| `/swordblock off` | Disable sword blocking |

Hold **right-click** with any sword to activate the blocking pose.
