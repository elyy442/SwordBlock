# SwordBlock
A Fabric client-side mod for Minecraft 1.21.10 and 1.21.11 that brings back the classic **1.8 sword blocking animation**.

## Features
* Works with any **ANIMATION** mods
* Works with any **TEXTURE** packs
* Added support for Hypixel 
* Added Arm Rotation on Third Person View

## Getting Started
Just type **/swordblock on/off** to enable/disable.
